Understanding Keyword Rules

Match All

ALL these keywords must be present. Like AND logic.

Example: ["BC", "450+"] means both "BC" AND "450+" must appear

Match Any Groups

For each group, AT LEAST ONE keyword must match. Like OR logic within each group.

Example: [["Branding","BC"]] means either "Branding" OR "BC" must appear

Exclude All

NONE of these keywords can be present. Like NOT logic.

Example: ["Digital", "UV"] means neither word can appear
