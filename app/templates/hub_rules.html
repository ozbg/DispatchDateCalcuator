Understanding Hub Selection Rules

Rule Application

Rules only remove hubs from the available list when constraints are not met

Example: If a hub exceeds size limits, it's removed as an option

Hub Selection

After applying rules, system tries to:

  1. Use current hub if still valid
  2. Fall back to remaining valid hubs

Constraints

  • Size and quantity limits
  • Equipment availability
  • Temporary restrictions
  • Product requirements

Matching Rules:

  • Required Keywords (AND Logic):
    ALL keywords must be present in the description.
    Example: "A4, Book, Jam" requires ALL three terms.
  • Excluded Keywords (OR Logic):
    ANY matching keyword will exclude the description.
    Example: If "spiral, wire" are excluded, a match to either will exclude.
  • Product IDs (OR Logic):
    ANY matching ID will include the product.
    Example: IDs "1,2,3" means the product must match ANY of these IDs.
